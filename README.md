# Comandos

### `npm start`

Inicia el proyecto en modo desarrollo.

### `npm test`

Ejecuta las pruebas del proyecto.

### `npm run build`

Genera una versión optimizada para producción.

### `npm run lint`

Verifica y muestra errores de estilo en el código.
